# Videos Folder

Place your video files in this folder to use them as local video sources.

## Usage

1. Drop your video files (`.mp4`, `.webm`, `.ogg`, etc.) into this folder
2. When creating a video, use the path: `/videos/your-filename.mp4`
3. The video will be served from `http://localhost:3000/videos/your-filename.mp4`

## Example

If you have a file named `sample-video.mp4` in this folder:
- Use the URL: `/videos/sample-video.mp4`
- The full URL will be: `http://localhost:3000/videos/sample-video.mp4`

## Supported Formats

- `.mp4` (H.264)
- `.webm`
- `.ogg`
- Any format supported by HTML5 video elements

## Note

Files in this folder are publicly accessible. Don't place sensitive or private content here.
